Utilisation du bouton central TrackPoint pour le défilement

 

  1. Placez le pointeur sur le document ou la page Web que vous souhaitez faire défiler.

     

  2. Maintenez le bouton central TrackPoint enfoncé. A l'écran, une icône de défilement remplace le pointeur pour indiquer que la fonction de défilement TrackPoint est activée.

     

  3. Pour faire défiler le document ou la page Web, maintenez le bouton central TrackPoint enfoncé et poussez le bouton de pointage TrackPoint dans le sens de votre choix. Vous pouvez faire défiler le fichier verticalement et horizontalement.

     

 

Remarque : Plus vous poussez le bouton de pointage TrackPoint, plus le défilement du document ou de la page Web est rapide. La fonction de défilement TrackPoint permet également de faire défiler le document ou la page Web directement en dessous du pointeur ŕ l'écran. Cela permet de faire défiler des documents ou des pages Web derričre la fenętre d'application active.
